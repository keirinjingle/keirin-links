選手DB … レース情報 …
「/」でスマート補完（選手／レース／戦法／タグ）。登録はボタンで行います（Enterでは登録しません）。Shift+Enterで改行。保存は端末ローカル＋（任意で）Google Driveに同期できます。
※「状態」が〔✅〕でないときは、レース情報補完は限定的になります。
データは端末ローカル（IndexedDB/localStorage）に保存。Google Drive同期を有効化した場合でも、あなたのDrive（App Folder）にのみ保存されます。

設定

※ 不特定多数への配布でも、開発者が発行した1つのクライアントIDを全ユーザーが共有して使います。利用者は自分のGoogleアカウントで初回のみログイン＆同意すればOKです。

→ ボタンを押すと、各ユーザーのブラウザでGoogleの同意画面が開きます（初回のみ）。

Google クライアントIDの取得ガイド

  1. Google Cloud Console → プロジェクト作成（例：mofu-memo）
  2. OAuth 同意画面：外部／承認済みドメインに github.io を追加／テストユーザーに自分を追加
  3. APIを有効化：Google Drive API
  4. スコープ：https://www.googleapis.com/auth/drive.appdata を追加（最小権限）
  5. クライアント → OAuth クライアント作成（種類：Webアプリ）
    • Authorized JavaScript origins：https://keirinjingle.github.io
    • Authorized redirect URIs：ポップアップ方式なら不要
  6. 発行された クライアントID を上の欄に貼って保存（シークレットは使いません）
